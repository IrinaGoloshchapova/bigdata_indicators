2017-09-07

Интенсивность инфляционных ожиданий

Индикатор показывает интенсивность ожиданий роста цен в краткосрочном периоде со стороны российского населения.

Индикатор представляет собой число комментариев пользователей сети Интернет к статьям в экономических СМИ по теме инфляции с упоминанием ожиданий роста цен, скорректированное на ожидания СМИ, число статей по теме инфляции в экономических СМИ с упоминанием ожиданий роста цен, в каждый период времени.
